# Odin Recipes

This is a simple project submission for The Odin Project to refresh and practice web development skills.
You can view the website on: [Odin Recipes](https://achill06.github.io/odin-recipes/)

## Project Overview

Odin Recipes is a small web application that lets users browse, create, edit, and delete recipes. The goal of this project is to practice core front-end and optionally back-end concepts: HTML, CSS, JavaScript, DOM manipulation, state management, responsive design, and persistence (localStorage or a simple backend).

## Features

- Browse a list of recipes
- View recipe details (ingredients, instructions)

## Technologies

- HTML5
